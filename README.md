# CodexBio.github.io

This repository now contains a simple single-file "link-in-bio" website scaffold (index.html, CSS and JS).

Quick notes
- Open the site locally: `python3 -m http.server 8000` then visit `http://localhost:8000`.
- Deploy on GitHub Pages: push `main`; in repo settings enable Pages from `main` branch (root).
- Edit `index.html` to change links, name, and the email in `js/script.js`.

Files added:
- `index.html` — main page
- `css/styles.css` — styles
- `js/script.js` — minimal interaction (theme + copy email)
- `favicon.svg` — avatar / favicon placeholder
